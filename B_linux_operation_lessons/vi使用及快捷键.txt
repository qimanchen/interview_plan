# vi
	- 基本用法
		a vi newfile.txt -- 利用vi打开一个文件
		b 输入vi -- 进入编辑状态
		c 编辑内容
		d esc -- 退出编辑模式
		e wq -- 保存并退出文件
# 特殊命令使用模式
	dd 删除单行
	ndd 删除从光标所在位置开始的n行
	u 返回上一次的内容（未保存:w）undo
	ctrl + r 撤销一次操作
	p 向下粘贴，3p将内容粘贴三次
	yy 复制光标所在行，3yy可以复制多行
	
	dG 把光标到行尾所有的内容删除
	
# 命令模式
	
	G 将光标切换到文件的最后一行
	gg 将光标切换到文件的第一行
	
	- 移动光标
		左右 h l
		上下 jk
		指定移动几行 n+G/gg
		gg 行首
		G 行尾
		$ 移动光标到行尾
		^/0 移动光标到行首
# 块选择模式
# 插入模式/编辑模式
	i 直接进入光标所在位置（光标所在字符的前面）进行编辑
	I 从光标所在行首开始进行编辑
	
	o 从光标所在位置的下一行进行编辑
	O 从光标所在位置的上一行进行编辑
	
	a 从光标所在位置的下一个字符（光标所在字符的后面）开始编辑
	A 从光标所在行的行尾进行编辑
	
	C 将光标到行尾的内容删除，并进入编辑模式
	cc 将整行内容进行删除，并进入编辑模式
	esc 退出编辑模式
# 低行模式
	: 进入底行模式
	/ 向下搜索(n向下，N向上）
	？向上搜索（n向上，N向下
	:q 直接退出
	:w 保存但不退出
	:wq 保存并退出
	:wq! 强制保存退出
	:q! 强制退出
	:set nu 显示行号
	esc 退出底行模式
# 特殊信息操作
	- 显示或取消行号
		:set nu
		:set nonu
	- 撤销操作
		u undo
		ctrl + r 撤销上次undo的操作
	- 快速搜索
		:set ic
		/search_string\c 忽略大小写
# 高级功能 - vim
	- 高亮显示
		搜索一个文件中没有的信息，取消高亮
	- 快速移动或复制信息
		:2,4move9 2到4行的信息，移动到第9行
		:2,4copy9 2到4行的信息，复制到第9行
	- 文件内容替换
		:%s#要替换的字符#新字符#g -- 将文件中内容信息全部提换
		s substitute
		g global 替换每行中所有字符，不加，则只替换第一个
		:2,4s#odl_str#new_str#g -- 2到4行内的信息替换
		:2,$s#old_str#new_string#g -- 2到末尾信息替换
	- 批量修改文件
		- 批量进行注释
			a 进入批量处理模式 ctrl +v
			b 选中需要加入的内容
			c shift+i 编辑内容
		- 批量删除信息
			a 进入批量处理模式 ctrl +v
			b 选中需要加入的内容
			c d/x
	- 替换
		r 单个字符替换
		R 持续替换
